MEAN Authentication Application

MEAN is a free and open-source JavaScript software stack for building dynamic web sites and web applications.[1] The MEAN stack is MongoDB, Express.js, AngularJS (or Angular), and Node.js. Because all components of the MEAN stack support programs are written in JavaScript, MEAN applications can be written in one language for both server-side and client-side execution environments.
